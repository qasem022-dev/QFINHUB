## Root Cause Diagnosis

**Google does not trust this site.** Despite technically sound SEO (A- grade), the site has:
- Zero off-site authority → crawl budget = zero → nothing gets indexed
- AI-generated content patterns → quality classifier flags → even "indexed" pages get zero clicks
- No real-world expertise signals → EEAT failure for YMYL finance niche

**The site is in a death spiral:** Low authority → no indexing → no clicks → no data to improve → content gets worse → Google trusts it less.
